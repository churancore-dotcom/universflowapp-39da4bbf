# Security Policy

## Supported Versions

Security updates are provided for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest | :x:               |

We always recommend running the latest released version of the app.

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue**.

Instead, email the maintainer directly:

**shashankyadavk12@gmail.com**

Please include:

- A clear description of the vulnerability
- Steps to reproduce it
- The potential impact
- Any suggested mitigation (optional)

### Response Time

The maintainer will acknowledge your report **within 72 hours** and will work
with you on a fix and coordinated disclosure timeline. Please give us a
reasonable window to patch the issue before any public disclosure.

Thank you for helping keep this project and its users safe.
